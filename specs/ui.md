# User Interface

## Header
- Load button
- Save button
- clear button

## Sidebar
- left side
- Seach bar
- filter interface
    - Filter by group
- After you search, it should show similar options (any search algorithm is fine as long as it is fast)

## Course Plan Visualizer
- right side, most of the page
- Each semester is a row
- Each semester starts with 5 empty slots laid out horizontally
- each course should be a rectangle with rounded corners

## Course Information Panel
- right of the course plan visualizer
- When a user clicks on a course, its information shows on this pane
- Add an "x" button to exit the panel

## Web support
- focus on large screen like laptops and desktops
